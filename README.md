Club Management System (C++):

A console-based Club Management System built using C++ that manages members, panels, events, and roles using file handling and OOP concepts.

Features:

Role-based login system (President, Advisor, GS, Executive, Normal)

Add, view, and delete members

Create and manage panels with assigned roles

Add and view events panel-wise

Secure role switching with password

Data stored using text files

Action logging system


Files Used:

members.txt – member information

panels.txt – Panel details

events.txt – Event records

log.txt – Activity logs

OOP Concepts Used:

Encapsulation,
Abstraction,
Polymorphism (logical),
File Handling.
